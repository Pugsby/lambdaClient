Pony Town Lambda Client is the first (and probably only) client that makes Pony Town look somewhat better.

## Build Instructions
```bash
sudo pacman -S git nodejs # Replace pacman with your package manager
git clone https://github.com/pugsby/lambdaClient.git
cd lambdaClient
npm run dist
# The built package is in ./dist
```
To install the built package:
``` bash
chmod +x ./install.sh
sudo ./install.sh
```
Flatpaks are coming eventually in order to make this easier.
